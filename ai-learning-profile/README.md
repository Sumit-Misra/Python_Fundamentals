This is a beginner Python project within my Python Fundamentals repository.

It generates an AI learning profile using only Python basics:
- input/output
- string methods
- number functions
- type conversion
- formatted output using f-strings

---

## What this project does

The script asks for:

- Name, age, country, profession  
- 3 key skills (comma-separated)  
- Study hours for the last 3 days  
- A performance score (0â€“100)  
- A short learning motto  
- Optional notes  

Then it:

- Cleans and formats input
- Uses string methods (`strip`, `title`, `lower`, `count`, `split`, `join`)
- Converts data types using `int`, `float`, `bool`
- Calculates total, average, min, max, and difference using:
  `round`, `abs`, `max`, `min`
- Prints a formatted learning report
